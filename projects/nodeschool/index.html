Npm


npm è un gestore di pacchetti per il linguaggio di programmazione JavaScript. È il gestore di pacchetti predefinito per l'ambiente di runtime JavaScript Node.js. Consiste in un client da linea di comando, chiamato anch'esso npm, e un database online di pacchetti pubblici e privati, chiamato npm registry. Wikipedia.

Soluzioni nodeschool.io

Per avvicinarmi al mondo Node.js ho iniziato dal sito nodeschool.io. Nel sito troverete una raccolta di esercizi interattivi (workshops) scaricabili tramite npm che pongono le basi per l'apprendimento di Node. In particolare vengono proposti esercizi in cui si viene introdotti agli stream, alla programmazione asincrona, all'uso dell'npm ecc... Se si è alle prime armi, in alcuni casi può essere difficile risolverli, per cui dare una sbirciatina alle soluzioni può essere d'aiuto! Qui sotto troverete le mie soluzioni ai vari moduli svolti:

Workshop Learnyounode:
Ottimo per prendere confidenza con la programmazione asincrona e le prime operazioni http.
Soluzioni learnyounode

Workshop stream adventure:
Uno dei punti forti che ho apprezzato fin da subito di node sono proprio gli stream, ecco il link alle soluzioni!
Soluzioni stream-adventure

Workshop how to npm:
Piccola introduzione all'npm, il grande raccoglietore di pacchetti di node. L'ho trovato davvero utile per fare le prime operazioni con npm; non pubblico le soluzioni di questo workshop in quanto non c'è codice da scrivere.
